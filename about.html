Yes, 100% free. No account, no subscription, no hidden charges. All 100 games are available to anyone with a web browser. The site is funded through standard display advertising.

Do the games work on phones and tablets?

Yes. All games were built with touch support. Most work in any modern mobile browser without any additional setup.

Yes. Every game on this site was developed in-house using HTML5 and JavaScript. Games inspired by classic formats (Tetris, Sudoku, etc.) are our own implementations, not copies of existing software.

Visit our Contact page to submit a game request or report any issues. We read every message.

Are the games safe for all ages?

Yes. All games are family-friendly and appropriate for all age groups. Many are educational — covering math, spelling, geography, and logic skills.
